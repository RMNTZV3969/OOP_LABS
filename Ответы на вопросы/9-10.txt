Контрольные вопросы: 

1. Как передать данные в шаблон? 
В метод render передать ключевые аргументы

2. Какие данные можно передать при рендеринге шаблона? 
Любые данные: строки, числа, списки, словари…

3. Как обозначается переменная в шаблоне? Как она связана с переменными в Python? 
В шаблоне переменная обозначается конструкцией {{}}. Она связана с переменной в пайтон тем, что тоже хранит какое-то значение.

4. Покажите конструкцию блока c условием в шаблоне. 
{% if name == 'Иван' %}
	{{ name }}
{% else %}
	Имя не Иван
{% endif %}

5. Покажите конструкцию блока с циклом шаблоне. 
{% for num in nums %}
	{{ num }}
{% endfor %}

6. Как создать свой фильтр для данных в шаблоне?
Нужно создать обычную функцию, например:
def datetimeformat(value, format='%H:%M / %d-%m-%Y'):
    return value.strftime(format)

а затем применить ее к переменной в шаблоне через символ |
{{ date | datetimeformat }}
