      3. Какие два типа циклов Вы знаете? Чем они отличаются?
Цикл for используется для итерации по последовательности, в то время как цикл while выполняется до тех пор, пока условие истинно.
В цикле for необходимо указать последовательность, по которой будет итерироваться цикл. В цикле while необходимо указать условие, при котором будет выполняться цикл.
Цикл for обычно используется, когда заранее известно число итераций, в то время как цикл while используется, когда число итераций заранее неизвестно.
      4. Для чего нужны функции print и input?
Print – вывод данных -  позволяет видеть результат выполнения программы, вывод сообщений, результатов вычислений и другой информации., input – ввод -  позволяет взаимодействовать с программой, вводить данные, значения или ответы на вопросы.
      5. На основе каких итерируемых объектов можно построить циклы?
1. Списки (list): Наиболее распространенный итерируемый объект. Вы можете создать цикл для перебора всех элементов в списке.
2. Кортежи (tuple): Подобно спискам, кортежи - это итерируемые объекты. Вы можете использовать цикл для перебора элементов кортежа.
3. Строки (str): В Python строки тоже являются итерируемыми объектами. Циклы могут использоваться для перебора символов строки.
4. Множества (set): Также могут использоваться в циклах, чтобы перебирать элементы множества
5. Словари (dict): Для перебора элементов словаря можно использовать ключи, значения или элементы (ключ, значение) в циклах.
      6. Какие Вы знаете встроенные функции? Назовите 3-5 шт.
1. print(): Используется для вывода информации на экран.
2. len(): Возвращает длину объекта, такую как строка, список, кортеж и т. д.
3. input(): Позволяет пользователю ввести данные с клавиатуры.
4. range(): Создает последовательность чисел в заданном диапазоне.
5. sorted(): Сортирует объект (например, список) в порядке возрастания (по умолчанию) или с помощью пользовательского ключа сортировки.

      8. Для чего нужны функции randint, choice, random в модуле random?
Функция randint(a, b) используется для генерации случайного целого числа в заданном диапазоне от a до b включительно.
Функция choice(seq) используется для выбора случайного элемента из последовательности seq, такой как список, кортеж или строка. 
Функция random() используется для генерации случайного числа с плавающей запятой в диапазоне от 0.0 до 1.0. Эта функция может быть использована для генерации случайных чисел в других диапазонах, умножая результат на соответствующий множитель.
