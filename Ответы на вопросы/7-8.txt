1. Какие модули и их функции используются для работы с данными в формате json?
Модуль json.
Функции модуля:
json.load - метод считывает файл в формате JSON и возвращает объекты Python
json.loads - метод считывает строку в формате JSON и возвращает объекты Python
json.dump - метод записывает объект Python в файл в формате JSON
json.dumps - метод возвращает строку в формате JSON


2. Какие модули и их функции используются для работы с данными в формате csv?
Модуль csv.
Чтение файла - Класс DictReader() модуля csv создает объект, который работает как обычный reader(), но отображает информацию о каждой строке в качестве словаря dict, ключи которого задаются необязательным параметром fieldnames.
Запись файла - Класс DictWriter() модуля csv создает объект, который работает как csv.writer(), но позволяет передавать строку с данными на запись как словарь, ключи которой задаются необязательным параметром fieldnames.

3. В каких режимах можно открыть файлы с помощью функции open()?
Если вы хотите прочесть файл - r
Для чтения и записи - r+
Для перезаписи содержимого файла - w
Добавление информации в файл - a
Для работы с двоичными файлами - b

6. Какой разделитель значений используется для формата csv? Можно ли его заменить?
При сохранении книги в виде файла . csv разделителем списка по умолчанию (разделителем) является запятая. Его можно изменить на другой символ разделителя с помощью параметров региона Windows.
